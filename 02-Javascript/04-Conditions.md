# Conditions

(https://developer.mozilla.org/fr/docs/Learn/JavaScript/Building_blocks/conditionals)

## if ... else

```js
if (ouvert){
    "faire ses achats";
}else{
    "revenir plus tard"
}

```

## Ternaire

```js
ouvert ? "faire ses achats" : "revenir plus tard"
```

## AND logical

```js
if (jour){
    "panneau solaire produit de l'élec"
}

//équivaut à 
jour && "panneau solaire produit de l'elec"
```
